The science of the soul: Doctrinal tendencies and clusters¶

This notebook contains the quantitative analyses of doctrinal tendencies and clusters on the possibility of having a science of the soul in the included commentaries.

NOTE: Identifying positive and non-positive groups of doctrines

During the anlysis we will seach for doctrines that are connected with their text as positive doctrines (= (:Question)-[:CONTAINS {type:'positive'}]->(d:Doctrine)). This is just a simpler way if identifying pieces of doctrine that are part of a constitutive part of the argument, and are thus also treated in the part of the chapter presenting positive doctrines.

This is the case for all doctrines that would be returned by this query:

MATCH (d:Doctrine)-[r]-(q:Question)--(:Text)--(a:Author)
WHERE (q)-[:INSTANCE_OF]->(:QuestionType {name:'Science of the soul'})
AND (d)-[:SUPPORTS|USES]-() 
AND NOT (d)-[:ATTACKS]-(:Doctrine {name:"There can be a science of the soul"})
OR NOT (d)--(:Doctrine)

This covers all doctrines on this specific question that have a edge of the type :SUPPORTS or :USES, either in- or out-going, and that does not have the edge type :ATTACKS to the root doctrine, or that may not have any edges to other :Doctrines.

This means that this also includes doctrines that can be used to attack an attack on the base doctrine (i.e. refute a negative ratio principalis), as long as that is not their only edge.

We are ready to start the analysis proper.

The main objectives are:

  1. Get some general information about the quantities, distribution and variety across the material.
  2. Identify clusters of commentators based on how many doctrines they share among each other.

General quantities¶

Let's look at some general quantities, both on all kinds of doctrines and within special types such as first level determination doctrines and rationes principales.

Note however that not all doctrines need be either a supporting or attacking doctrine of a specific other doctrine. The obvious example is of course a root doctrine such as the basic positive idea that there can be a science about the soul, but some doctrines also describe a general tendency that can occur across a range of different discussions (e.g. the distinction between proper and improper science).

def commentator_doctrines(name: str) -> List[int]:
    q_all = ("MATCH (root:Doctrine {name:'There can be a science of the soul'})"
             "MATCH (a:Author)--(:Text)--(q:Question)--(d:Doctrine)"
             "WHERE (q)-[:INSTANCE_OF]->(:QuestionType {name:'Science of the soul'})"
             "AND a.name = '%s'"
             "RETURN count(distinct d)" % name)
    q_det = ("MATCH (root:Doctrine {name:'There can be a science of the soul'})"
             "MATCH (a:Author)--(:Text)--(q:Question)--(d:Doctrine)"
             "WHERE (d)-[:SUPPORTS]-(root)"
             "AND a.name = '%s'"
             "RETURN count(distinct d)" % name)
    q_pos = ("MATCH (a:Author)--(:Text)--(q:Question)-[:CONTAINS {type:'positive'}]->(d:Doctrine)"
             "WHERE (q)-[:INSTANCE_OF]->(:QuestionType {name:'Science of the soul'})"
             "AND a.name = '%s'"
             "RETURN count(distinct d)" % name)
    q_att = ("MATCH (root:Doctrine {name:'There can be a science of the soul'})"
             "MATCH (a:Author)--(:Text)--(q:Question)--(d:Doctrine)"
             "WHERE (d)-[:ATTACKS]->(root)"
             "AND a.name = '%s'"
             "RETURN count(distinct d)" % name)
    q_ref = ("MATCH (root:Doctrine {name:'There can be a science of the soul'})"
             "MATCH (a:Author)--(:Text)--(q:Question)--(d:Doctrine)"
             "WHERE (d:Doctrine)-[:ATTACKS]->(:Doctrine)-[:ATTACKS]->(root)"
             "AND a.name = '%s'"
             "RETURN count(distinct d)" % name)
    q_free = ("MATCH (a:Author)--(:Text)--(q:Question)--(d:Doctrine)"
              "WHERE (q)-[:INSTANCE_OF]->(:QuestionType {name:'Science of the soul'})"
              "AND NOT (d)--(:Doctrine)"
              "AND a.name = '%s'"
              "RETURN count(distinct d)" % name)
    res_all = run_query(q_all).single()[0]
    res_det = run_query(q_det).single()[0]
    res_pos = run_query(q_pos).single()[0]
    res_att = run_query(q_att).single()[0]
    res_ref = run_query(q_ref).single()[0]
    res_free = run_query(q_free).single()[0]
    return [res_det, res_free, res_pos, res_att + res_ref, res_att, res_ref, res_all]

def unique_shared_determinations_all(authors):
    d = {}
    for a in authors:
        unique = count_unique_determination_doctrines(a)

        prop = unique_to_shared_proportion(unique, shared)

    return d

def number_shared_positive_doctrines(name):

def proportion_shared_positive_doctrines(name):
    q = ("MATCH (d)-[{type:'positive'}]-(q:Question)--(:Text)--(a1:Author {name:'%s'})"
         "WHERE (q)-[:INSTANCE_OF]->(:QuestionType {name:'Science of the soul'})"
         "MATCH (d)-[{type:'positive'}]-(:Question)--(:Text)--(a2:Author)"
         "MATCH (d1)-[{type:'positive'}]-(q1:Question)--(:Text)--(a1)"
         "WHERE (q1)-[:INSTANCE_OF]->(:QuestionType {name:'Science of the soul'})"
         "MATCH (d2)-[{type:'positive'}]-(q2:Question)--(:Text)--(a2)"
         "WHERE (q2)-[:INSTANCE_OF]->(:QuestionType {name:'Science of the soul'})"
         "RETURN a2.name, (toFloat(count(distinct d)) / count(distinct d1)) * (toFloat(count(distinct d)) / count(distinct d2))" % name)
    res = run_query(q)
    return {r[0]: r[1] for r in res}

def build_matrix(authors, docfunc=proportion_shared_positive_doctrines):
    matrix = {}
    for author in authors:
        matrix[author] = []
        values = docfunc(author)
        for name in authors:
            if name in values:
                matrix[author].append(values[name])
            else:
                matrix[author].append(0)
    return matrix

d = build_matrix(author_names)

def proportion_shared_negative_doctrines(name):
    q = ("MATCH (d:Doctrine)-[:ATTACKS *1..2]->(:Doctrine {name:'There can be a science of the soul'})"
         "MATCH (a:Author {name:'%s'})--(:Text)--(q1:Question)--(d)"
         "MATCH (d)--(q2:Question)--(:Text)--(a2:Author)"
         "MATCH (q1)--(d1:Doctrine) WHERE (d1)-[:ATTACKS *1..2]->(:Doctrine {name:'There can be a science of the soul'})"                         
         "MATCH (q2)--(d2:Doctrine) WHERE (d2)-[:ATTACKS *1..2]->(:Doctrine {name:'There can be a science of the soul'})"                         
         "RETURN a2.name, (toFloat(count(distinct d)) / count(distinct d1)) * (toFloat(count(distinct d)) / count(distinct d2))" % name)
    res = run_query(q)
    return {r[0]: r[1] for r in res}

Binary cluster map¶

Here we want to create a matrix table where it can be seen in more detail which doctrines each commentator uses.

I have used this as an inspiration: https://stats.stackexchange.com/questions/222492/any-distance-measures-that-are-more-useful-for-binary-data-clustering. As suggested in the post we use the Hamming metric for distance measuring, as that just counts the number of coordinates that differ. AS it is also mentioned in the scipy documentation it can therefore be used for boolean matrices (https://docs.scipy.org/doc/scipy/reference/generated/scipy.spatial.distance.pdist.html).

def create_matrix(authors, calculator):
